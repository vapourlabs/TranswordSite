Commands List

Wanting to learn more about how Transword's bot commands work? Well, look no more! Down below is a list of all of the available commands by default given when you invite Transword for your very own server.




Name Category Arguments Description
/help Utilities | OPTIONAL name Returns an embed showing the bot's commands.
/ping Utilities Returns the bot's latency as milliseconds.
/about Utilities Provides an embed showing information about the bot.
/vote Utilities Provides a website link to vote for Transword's reputability.
/translate Translation target text | OPTIONAL formality format Translates text from a foreign language into another one specified.
/transauto Translation target Toggles an automatic translation state of message deletion to convert to another language specified.
/stats Translation Provides a list of statistics for the bot's usage.
